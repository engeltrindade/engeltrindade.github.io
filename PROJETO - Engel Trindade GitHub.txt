CONHECIMENTOS 

* ETL 

Do inglês Extract Transform Load, são ferramentas de software cuja função é a extração de dados de diversos sistemas, transformação desses dados conforme regras de negócio e por fim o carregamento dos dados.

* DAX
Expressões de análise de dados é a fórmula nativa e linguagem de consulta para modelos de tabela Microsoft Power BI.

* SQL 
 Sistema gerenciador de Banco de dados relacional. 

* STORYTELLING 
O Storytelling é a arte de saber contar “histórias” com dados de forma fluida e organizada! Com o objetivo de fazer com que o público-alvo tenha uma fácil leitura da sua página, relatório, dashboard, etc e entenda exatamente a história que você quer contar!

* FIGMA 
Editor gráfico de vetor e prototipagem de projetos de design, utilizado na criação do background (design) dos dashboard's. 

* LANGUAGE M 
Responsável por todo o processo de transformação realizado no Power BI e no suplemento Power Query.
